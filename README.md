# xui
A lightweight C++ DirectUI framework for Windows desktop application development。

xui是一款用C++开发的轻量级DirectUI框架，适用于Windows桌面程序开发。

# 使用要求：
* Windows 7及以上操作系统
* 支持C++ 11的MSVC编译器
* 包含ATL和GDI+的Windows SDK

# 目标平台
* Windows 7及以上操作系统，x86和x64架构
